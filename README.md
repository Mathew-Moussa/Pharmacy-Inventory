# Pharmacy Inventory System 🏥

A cloud-native inventory management system built with Node.js, MongoDB, and Nginx, deployed on AWS using Docker.

## 🚀 How to Run Locally (Docker)

To run the entire stack on your local machine, ensure you have **Docker** and **Docker Compose** installed, then follow these steps:

1. **Clone the project files** to your local directory.
2. **Open your terminal** in the project root folder.
3. **Run the following command:**
   ```bash
   docker-compose up --build
4. **Access the app:**

Frontend: http://localhost:80

Backend API: http://localhost/api/medicines
